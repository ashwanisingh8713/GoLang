## Keeps Coupon data
## Coupon T&C will not be part of this because coupon must be identical
## T&C can be created with different table with reference of coupon_id.


coupon_id   uuid        (PK)
coupon_code text
coupon_desc text
created_at  timestamp
created_date date       (CK)
modified_at timestamp
expired_at  timestamp
is_active   boolean     (CK)
